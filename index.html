Secure Telemedicine Web Application for ECG Monitoring & Analytics

Course: Tecnologie per la Telemedicina

Developed a full-stack, GDPR-compliant web application utilizing Streamlit for the secure remote upload, monitoring, and analysis of multi-lead electrocardiogram (ECG) data.

Key Contributions & Engineering Pillars:
  • Interactive Visualization Front-end: Engineered a high-performance interactive interface simulating millimeter graph paper, allowing clinicians to zoom, filter specific anatomical leads, and seamlessly navigate continuous time-series ECG data.
  • Digital Signal Processing (DSP) Engine: Implemented mathematical algorithms to ingest raw electrical signals, execute automated R-peak detection, compute continuous R-R intervals, and extract instantaneous Heart Rate (HR) metrics.
  • Data Security & GDPR Vault: Applied 'Privacy by Design' via patient identity pseudonymization, built a custom encryption-at-rest system using Linear Feedback Shift Registers, and integrated SHA-256 cryptographic hashing to ensure absolute data integrity.
  • Cryptographic Authentication: Conceptualized and simulated a DH-EKE protocol as an independent Python security engine, guaranteeing zero-knowledge password transmission to eliminate network interception risks.

Neuronal Network Activity Analysis Across Development & Pharmacology

Course: Neuroengineering

Engineered a unified, high-throughput Python data pipeline to characterize extracellular spontaneous activity and pharmacological modulation in human pluripotent stem cell-derived (hPSC) cortical networks cultured on Microelectrode Arrays (MEAs).

Key Contributions & Bio-Data Insights:
  • Signal Processing Pipeline: Developed a robust computational pipeline using NumPy and SciPy to perform 4th-order Butterworth band-pass filtering (300-3000 Hz) and Common Median Referencing (CMR) to isolate single-unit spike events from Local Field Potentials (LFPs).
  • Iterative Spike & Burst Sorting: Implemented MAD-based thresholding and an Interspike Interval (ISI) burst detection algorithm, validating classifications against ground-truth datasets.
  • Relative Similarity Matrices: Programmed pairwise statistical comparisons across natural developmental stages (DIV 3-35) and mature drug conditions (DIV 29 with GABA, Gabazine, CNQX, D-AP5, Kainic Acid, and TTX) utilizing normalized similarity indices.
  • Systems-Level Analysis: Demonstrated through programmatic data visualizations (Seaborn/Matplotlib) that numerical similarities between drug-constrained configurations and early development do not imply functional biological equivalence.

Microfluidic Hair Follicle-on-a-Chip for Preclinical Minoxidil Screening

Course: Frontiers in Bioengineering enabling nanotechnologies

Designed and modeled a state-of-the-art commercial single-organ-on-chip (OoC) platform engineered to simulate dihydrotestosterone (DHT)-induced androgenetic alopecia (AGA) and evaluate the therapeutic efficacy of Minoxidil under dynamic microenvironments.

Key Contributions & Bio-Design Innovations:
  • Advanced Thermoplastic Architecture: Utilized Cyclo Olefin Copolymer (COC) and Polyethylene Terephthalate (PET/BioPET 002) to build a multi-compartment device, significantly minimizing hydrophobic drug absorption compared to legacy PDMS platforms to guarantee true dose-response gradients.
  • 3D Epithelial-Mesenchymal Blueprint: Structured a spatially organized 3D construct integrating human dermal papilla cell (IDP) spheroids, outer root sheath (ORS) keratinocytes, and dermal fibroblasts embedded in a remodeling Type I collagen matrix under Air-Liquid Interface (ALI) conditions.
  • Open-Circuit One-Pass Perfusion: Conceptualized a continuous fluidic circulation strategy that closely mimics systemic vascular nutrient supply and hormone delivery while preventing metabolite waste accumulation.
  • Real-Time Non-Invasive Analytics: Integrated dual-wavelength fluorescence monitoring (YFP/CFP ratio tracking) for real-time tracking of canonical Wnt/β-catenin signaling pathway reactivation alongside trichoscopic morphological assessment criteria.

Automated CAD System for Colorectal Cancer vs. Diverticular Disease Differentiation

Course: Data Science in Healthcare

Developed an AI-driven Computer-Aided Diagnosis (CAD) system in Python to automatically differentiating between colorectal cancer and diverticular disease from 3D Computed Tomography (CT) scans, targeting the reduction of misdiagnoses and invasive procedures.

Challenges Solved:
  • 3D Medical Data Quality Control: Managed volumetric NIfTI (.nii) files using `nibabel`. Built a preprocessing filter to detect and isolate corrupted profiles, handling spatial resolution mismatches between CT scans and manual masks by labeling anomalies as NaN to secure clean data ingestion.
  • Cross-Center Validation & Bias Control: Analyzed statistical feature distributions between two independent hospital databases. Proved data homogeneity across clinical sites to ensure that scanner calibration differences wouldn't introduce institutional bias.
  • Inter-Observer Variability Engineering: Addressed the core challenge of conflicting manual physician segmentations.
  • Clinical Generalizability Testing: Maintained strict separation of clinical cohorts by reserving one entire database from one of the centers as an unseen external Test Set, verifying the model's accuracy and adaptability in independent hospital environments.
